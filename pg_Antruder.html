Antruder

About

Antruder is a free and open source software developed to prevent unauthorised USB drives from being accessed by the host machine by use of a password.

How to download

  • Scroll down to the bottom of the page till you find the 'Download' button.
  • Click on the 'Download' button (Your download should begin shortly).
  • After downloading has completed, extract the compressed zip file to any location of your choice. If you do not have an extracting software you can get one here.
  • Run the extracted setup file with administrator priviledges (You may need to disable your Anti-virus software to be able to run the software).
  • Finish the installation process by moslty clicking Next untill the installation process is complete.
  • Please take note that the default user password for Antruder is '0000' without quotes, this makes the system vulnerable to users who are friendly to Antruder (the ones that have used it before) so we highly recommend that you change it soon after installation process.

Requirements

NOTE: Before downloading and installing Antruder, we recommend you to check the app's system requirements below.

  • Windows Operating System - 7,8,8.1,10 & 11 (x86/x64).
  • At least 1GB operating Ram.
  • This version of Antruder uses .Net Framework 3.5 which you can download from here
  • 10MB HDD space or above.
